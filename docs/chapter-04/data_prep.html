World Development Indicator Dashboard¶

World Development Indicators (WDI) is the World Bank’s premier compilation of cross-country comparable data on development. This notebook prepares the data for producing interactive visualizations.

References

  • http://wdi.worldbank.org/

  • https://wbdata.readthedocs.io/en/stable/

  • https://datahelpdesk.worldbank.org/knowledgebase/topics/125589-developer-information

  • https://databank.worldbank.org/source/world-development-indicators

Data Source

  • https://datacatalog.worldbank.org/dataset/world-development-indicators

  • https://unstats.un.org/unsd/methodology/m49/overview
